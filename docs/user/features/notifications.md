# Notifications

Stay informed with real-time notifications about important events and updates.

## Types of Notifications

### In-App Notifications

Appear within the application:
- Bell icon in the header
- Badge count for unread items
- Notification panel with details

### Push Notifications

Delivered to your device:
- Browser notifications (web)
- Mobile push notifications (Android/iOS)
- Available even when app is closed

### Email Notifications

Sent to your registered email:
- Important updates
- Weekly digests
- Account alerts

## Viewing Notifications

### Notification Center

1. Click the bell icon in the header
2. View list of recent notifications
3. Click any notification to view details
4. Mark as read or dismiss

### Notification Details

Each notification shows:
- **Title** - Brief summary
- **Message** - Full details
- **Time** - When it was sent
- **Action** - Link to related content

## Managing Notifications

### Mark as Read

- **Single**: Click the notification
- **All**: Click "Mark all as read"

### Dismiss Notifications

- **Single**: Click the X or swipe away
- **All**: Click "Clear all"

### Archive

Keep notifications for reference without cluttering your inbox.

## Push Notification Settings

### Enabling Push Notifications

**Web (Browser)**:
1. When prompted, click "Allow" for notifications
2. Or go to browser settings to enable

**Mobile**:
1. Allow notifications when installing the app
2. Or enable in device Settings > Apps > IDICO IPMS

### Disabling Push Notifications

**Web**:
1. Go to **Settings > Notifications**
2. Toggle off "Push Notifications"

**Mobile**:
1. Device Settings > Apps > IDICO IPMS
2. Turn off Notifications

## Notification Preferences

### Categories

Choose which notifications you receive:

| Category | Description |
|----------|-------------|
| **System** | Account and security alerts |
| **Updates** | Feature and content updates |
| **Reminders** | Task and deadline reminders |
| **Marketing** | News and announcements |

### Frequency

Control how often you receive notifications:

- **Real-time** - Immediately when event occurs
- **Digest** - Daily or weekly summary
- **Off** - No notifications for this category

### Quiet Hours

Set times when notifications are silenced:

1. Go to **Settings > Notifications > Quiet Hours**
2. Enable quiet hours
3. Set start and end times
4. Save changes

Notifications will be queued and delivered after quiet hours.

## Troubleshooting

### Not Receiving Notifications

**Check app settings**:
1. Go to **Settings > Notifications**
2. Ensure notifications are enabled
3. Check category preferences

**Check device settings**:
- Browser: Allow notifications for the site
- Mobile: Enable notifications in device settings

**Check connection**:
- Ensure you have internet connectivity
- Try logging out and back in

### Too Many Notifications

1. Go to notification preferences
2. Disable non-essential categories
3. Switch to digest mode for less urgent updates
4. Set up quiet hours

### Notification Sound Issues

**Web**:
- Check browser and system sound settings
- Ensure Do Not Disturb is off

**Mobile**:
- Check app notification sound settings
- Verify device is not on silent mode

## Best Practices

1. **Enable important categories** - Stay informed about critical updates
2. **Use quiet hours** - Prevent disturbances during off-hours
3. **Review regularly** - Clear old notifications
4. **Customize per device** - Different settings for work/personal
