# Dashboard Overview

The dashboard is your central hub for accessing all IDICO IPMS features.

## Dashboard Layout

### Navigation Sidebar

Located on the left side of the screen, the sidebar provides quick access to:

- **Dashboard** - Main overview page
- **Customers** - Customer management
- **Contracts** - Contract information
- **Industrial Parks** - Park details and management
- **Service Requests** - View and manage requests
- **Settings** - Personal preferences

On mobile devices, tap the menu icon to reveal the sidebar.

### Header

The top header includes:

- **Search** - Find content quickly
- **Notifications** - View recent alerts
- **Profile Menu** - Access profile and settings

### Main Content Area

The central area displays:

- Key metrics and statistics
- Recent activity
- Quick action buttons
- Important notifications

## Navigation

### Using the Sidebar

1. Click on any menu item to navigate
2. Active page is highlighted
3. Submenus expand on click

### Breadcrumbs

Shows your current location in the app hierarchy:
```
Dashboard > Customers > Customer Details
```

Click any breadcrumb to navigate back.

### Quick Actions

Access common tasks from the dashboard:
- View recent items
- Create new records
- Access reports

## Dashboard Widgets

### Overview Cards

Display key metrics:
- Total customers
- Active contracts
- Pending requests
- Recent activity

### Activity Feed

Shows recent events:
- New registrations
- Updated records
- System notifications

## Responsive Design

The dashboard adapts to your device:

| Device | Layout |
|--------|--------|
| Desktop | Full sidebar, expanded widgets |
| Tablet | Collapsible sidebar, adjusted widgets |
| Mobile | Hidden sidebar, stacked widgets |

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `/` | Open search |
| `?` | Show help |
| `Esc` | Close dialog |

## Customization

Personalize your dashboard experience:

1. **Theme** - Switch between light and dark mode
2. **Language** - Change display language
3. **Notifications** - Configure alert preferences

See [Settings](/docs/user/features/settings) for more options.

## Tips

- **Pin frequently used pages** for quick access
- **Use search** to find anything quickly
- **Check notifications** regularly for updates
- **Bookmark the dashboard** for easy access
