# Welcome to IDICO IPMS

IDICO IPMS (Industrial Park Management System) is a comprehensive platform designed to streamline industrial park operations and enhance communication between administrators and stakeholders.

## What is IDICO IPMS?

IDICO IPMS provides a modern, user-friendly interface for:

- **Dashboard** - View key metrics and notifications at a glance
- **Profile Management** - Update your personal information and preferences
- **Notifications** - Stay informed with real-time push notifications
- **Settings** - Customize your experience with language and theme options

## Platform Availability

IDICO IPMS is available on multiple platforms:

| Platform | Access |
|----------|--------|
| **Web** | Access via any modern web browser |
| **Android** | Download from Google Play Store |
| **iOS** | Download from Apple App Store |
| **PWA** | Install directly from your browser |

All platforms share the same features and sync your data in real-time.

## Key Features

### Multi-Language Support
Switch between English and Vietnamese at any time.

### Dark Mode
Choose between light and dark themes for comfortable viewing.

### Push Notifications
Receive timely updates about important events and announcements.

### Offline Support
Access cached content even when offline (PWA feature).
