# Troubleshooting

Solutions to common issues you may encounter with IDICO IPMS.

## Login Problems

### Can't Log In

**Symptoms**: Login button doesn't work, error message appears

**Solutions**:
1. Check your email and password are correct
2. Ensure Caps Lock is off
3. Clear browser cache and cookies
4. Try a different browser
5. Reset your password if needed

### "Invalid Credentials" Error

**Cause**: Incorrect email or password

**Solutions**:
1. Double-check your email address
2. Verify your password (try typing it in a visible field first)
3. Use the "Forgot Password" link to reset
4. Contact your administrator if issue persists

### Account Locked

**Cause**: Too many failed login attempts

**Solutions**:
1. Wait 15-30 minutes before trying again
2. Use "Forgot Password" to reset
3. Contact your administrator for immediate unlock

### Google Login Not Working

**Solutions**:
1. Ensure you're using the correct Google account
2. Check if pop-ups are blocked (allow pop-ups for the site)
3. Clear cookies and try again
4. Try incognito/private browsing mode

## App Performance

### Slow Loading

**Solutions**:
1. Check your internet connection
2. Clear browser cache: **Settings > Clear browsing data**
3. Disable browser extensions temporarily
4. Try a different network
5. Close other tabs/applications

### App Not Responding

**Solutions**:
1. Refresh the page (Ctrl+R or Cmd+R)
2. Clear browser cache
3. Close and reopen the browser
4. Try a different browser

### Features Not Working

**Solutions**:
1. Refresh the page
2. Clear cache and cookies
3. Check for browser updates
4. Disable ad blockers or privacy extensions
5. Try incognito mode

## Display Issues

### Page Layout Broken

**Solutions**:
1. Zoom to 100% (Ctrl+0 or Cmd+0)
2. Clear cache
3. Disable browser extensions
4. Try a different browser

### Text Not Displaying Correctly

**Solutions**:
1. Check language settings
2. Refresh the page
3. Clear browser cache
4. Ensure your browser supports the language

### Dark Mode Issues

**Solutions**:
1. Toggle theme off and on in Settings
2. Check system preferences (if using System theme)
3. Clear cache
4. Try a different browser

## Notification Issues

### Not Receiving Notifications

**Browser Notifications**:
1. Check browser permissions for the site
2. Ensure notifications aren't blocked
3. Check if Do Not Disturb is enabled

**Mobile Notifications**:
1. Check app notification settings in device Settings
2. Ensure app has notification permission
3. Check battery optimization isn't blocking the app

**In-App**:
1. Go to Settings > Notifications
2. Ensure notifications are enabled
3. Check specific category settings

### Too Many Notifications

**Solutions**:
1. Go to **Settings > Notifications**
2. Disable non-essential categories
3. Set up Quiet Hours
4. Switch to digest mode for less urgent notifications

## Mobile App Issues

### App Crashes

**Solutions**:
1. Force close and reopen the app
2. Clear app cache (device Settings > Apps > IDICO IPMS)
3. Update to the latest version
4. Restart your device
5. Reinstall the app

### Can't Download/Install

**Solutions**:
1. Check available storage space
2. Ensure you have internet connection
3. Try using WiFi instead of mobile data
4. Clear Play Store/App Store cache
5. Sign out and back into your store account

### Sync Issues

**Solutions**:
1. Check internet connection
2. Pull down to refresh
3. Log out and log back in
4. Clear app cache
5. Reinstall the app

## Data Issues

### Data Not Saving

**Solutions**:
1. Check internet connection
2. Look for error messages
3. Try again after a few moments
4. Refresh and retry
5. Contact support if issue persists

### Data Appears Missing

**Solutions**:
1. Refresh the page
2. Check filters that might be hiding data
3. Clear cache and reload
4. Log out and back in
5. Contact support to verify data status

### Sync Conflicts

**Solutions**:
1. Refresh to get latest data
2. If editing, save your changes to a note first
3. Reload and apply changes again
4. Contact support for complex conflicts

## Contact Support

If you've tried the above solutions and still have issues:

### Information to Provide

When contacting support, include:
- Description of the problem
- Steps to reproduce
- Screenshots if applicable
- Browser and version
- Device and operating system
- Error messages (exact text)

### Support Channels

- **Email**: Contact your system administrator
- **In-App**: Settings > Support > Report Issue
- **Documentation**: Check [FAQ](/docs/user/faq) for more solutions
