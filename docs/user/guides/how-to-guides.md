# How-To Guides

Step-by-step instructions for common tasks in IDICO IPMS.

## Account Management

### How to Change Your Email

1. Contact your system administrator
2. Verify your identity
3. Administrator will update your email
4. You'll receive confirmation at your new email

### How to Reset Your Password

1. Go to the login page
2. Click **Forgot Password?**
3. Enter your email address
4. Check your email for a reset link
5. Click the link and create a new password
6. Log in with your new password

### How to Enable Two-Factor Authentication

1. Go to **Settings > Security**
2. Click **Enable 2FA**
3. Scan the QR code with your authenticator app
4. Enter the verification code
5. Save your backup codes securely
6. Click **Confirm**

## Profile Customization

### How to Update Your Profile Picture

1. Click your avatar in the header
2. Select **Profile**
3. Click the camera icon on your avatar
4. Choose an image from your device
5. Crop if needed
6. Click **Save**

### How to Change Display Language

1. Go to **Settings > Language**
2. Select your preferred language
3. Click **Save**
4. Interface updates immediately

### How to Switch to Dark Mode

1. Go to **Settings > Appearance**
2. Select **Dark** theme
3. Or toggle the theme switcher in the header

## Notifications

### How to Enable Push Notifications

**On Web (Browser):**
1. When prompted, click **Allow**
2. If you missed the prompt, go to browser settings
3. Find site permissions > Notifications
4. Set to **Allow**

**On Mobile:**
1. Open device Settings
2. Go to Apps > IDICO IPMS
3. Enable Notifications
4. Choose notification types to allow

### How to Mute Notifications

1. Go to **Settings > Notifications**
2. Enable **Quiet Hours**
3. Set your preferred start and end times
4. Click **Save**

### How to Customize Notification Types

1. Go to **Settings > Notifications**
2. Review notification categories
3. Toggle each category on/off
4. Set frequency (real-time, digest, off)
5. Click **Save**

## Mobile App

### How to Install the Mobile App

**Android:**
1. Open Google Play Store
2. Search for "IDICO IPMS"
3. Tap **Install**
4. Open the app and log in

**iOS:**
1. Open App Store
2. Search for "IDICO IPMS"
3. Tap **Get**
4. Open the app and log in

### How to Install as PWA (Progressive Web App)

**Chrome:**
1. Visit the IDICO IPMS website
2. Click the install icon in the address bar
3. Click **Install**

**Safari (iOS):**
1. Visit the IDICO IPMS website
2. Tap the Share button
3. Select **Add to Home Screen**
4. Tap **Add**

### How to Enable Offline Mode

1. Make sure you're logged in
2. Visit pages you want available offline
3. They'll be cached automatically
4. Access them even without internet

## Data Management

### How to Export Your Data

1. Go to **Settings > Privacy > Data**
2. Click **Export My Data**
3. Choose format (JSON or CSV)
4. Click **Export**
5. Download the file when ready

### How to Clear Cached Data

1. Go to **Settings > Data & Storage**
2. Click **Clear Cache**
3. Confirm the action
4. Cache is cleared immediately

## Security

### How to View Login History

1. Go to **Settings > Security**
2. Click **Login History**
3. Review recent logins
4. Look for unfamiliar devices or locations

### How to Sign Out of All Devices

1. Go to **Settings > Security > Sessions**
2. Review active sessions
3. Click **Sign Out All Devices**
4. Confirm the action
5. You'll need to log in again on all devices

### How to Report a Security Concern

1. Go to **Settings > Security**
2. Click **Report Issue**
3. Describe the concern
4. Submit the report
5. You'll receive a response from support
