# Tổng quan Bảng điều khiển

Bảng điều khiển là trung tâm của bạn để truy cập tất cả các tính năng IDICO IPMS.

## Bố cục Bảng điều khiển

### Thanh bên Điều hướng

Nằm ở bên trái màn hình, thanh bên cung cấp truy cập nhanh đến:

- **Bảng điều khiển** - Trang tổng quan chính
- **Khách hàng** - Quản lý khách hàng
- **Hợp đồng** - Thông tin hợp đồng
- **Khu Công nghiệp** - Chi tiết và quản lý khu
- **Yêu cầu Dịch vụ** - Xem và quản lý yêu cầu
- **Cài đặt** - Tùy chọn cá nhân

Trên thiết bị di động, chạm biểu tượng menu để hiển thị thanh bên.

### Thanh tiêu đề

Thanh tiêu đề phía trên bao gồm:

- **Tìm kiếm** - Tìm nội dung nhanh chóng
- **Thông báo** - Xem các cảnh báo gần đây
- **Menu Hồ sơ** - Truy cập hồ sơ và cài đặt

### Vùng Nội dung chính

Vùng trung tâm hiển thị:

- Các chỉ số và thống kê quan trọng
- Hoạt động gần đây
- Các nút hành động nhanh
- Thông báo quan trọng

## Điều hướng

### Sử dụng Thanh bên

1. Nhấp vào bất kỳ mục menu nào để điều hướng
2. Trang đang hoạt động được tô sáng
3. Menu con mở rộng khi nhấp

### Đường dẫn (Breadcrumbs)

Hiển thị vị trí hiện tại của bạn trong hệ thống phân cấp ứng dụng:
```
Bảng điều khiển > Khách hàng > Chi tiết Khách hàng
```

Nhấp vào bất kỳ đường dẫn nào để quay lại.

### Hành động nhanh

Truy cập các tác vụ phổ biến từ bảng điều khiển:
- Xem các mục gần đây
- Tạo bản ghi mới
- Truy cập báo cáo

## Các Widget Bảng điều khiển

### Thẻ Tổng quan

Hiển thị các chỉ số quan trọng:
- Tổng số khách hàng
- Hợp đồng đang hoạt động
- Yêu cầu đang chờ
- Hoạt động gần đây

### Dòng Hoạt động

Hiển thị các sự kiện gần đây:
- Đăng ký mới
- Bản ghi được cập nhật
- Thông báo hệ thống

## Thiết kế Đáp ứng

Bảng điều khiển thích ứng với thiết bị của bạn:

| Thiết bị | Bố cục |
|----------|--------|
| Máy tính | Thanh bên đầy đủ, widget mở rộng |
| Máy tính bảng | Thanh bên có thể thu gọn, widget điều chỉnh |
| Di động | Thanh bên ẩn, widget xếp chồng |

## Phím tắt

| Phím tắt | Hành động |
|----------|-----------|
| `/` | Mở tìm kiếm |
| `?` | Hiển thị trợ giúp |
| `Esc` | Đóng hộp thoại |

## Tùy chỉnh

Cá nhân hóa trải nghiệm bảng điều khiển của bạn:

1. **Giao diện** - Chuyển đổi giữa chế độ sáng và tối
2. **Ngôn ngữ** - Thay đổi ngôn ngữ hiển thị
3. **Thông báo** - Cấu hình tùy chọn cảnh báo

Xem [Cài đặt](/docs/user/features/settings) để biết thêm tùy chọn.

## Mẹo

- **Ghim các trang thường dùng** để truy cập nhanh
- **Sử dụng tìm kiếm** để tìm bất cứ thứ gì nhanh chóng
- **Kiểm tra thông báo** thường xuyên để cập nhật
- **Đánh dấu bảng điều khiển** để truy cập dễ dàng
